## 它能做什么

你只需要在电脑微信里打开「跳一跳」小游戏，剩下的交给程序：

- 自动截取游戏画面
- 识别棋子位置（紫色小人）
- 识别下一个跳台位置
- 计算需要按多久鼠标
- 模拟鼠标长按完成跳跃
- 全程自动循环，帮你刷高分

有可视化界面，能实时看到识别结果（红框是棋子、蓝框是目标、绿线是跳跃路径），参数随时可调。

## 从零开始：一步一步教程

### 1. 安装 Python

如果电脑上已经装过 Python（在命令行输入 `python --version` 能看到版本号），可以跳过这一步。

**去官网下载：**

打开 https://www.python.org/downloads/ ，点击黄色的大按钮下载最新版。

**安装时注意：**

- 一定要 **勾选底部的「Add Python to PATH」**（这一步很重要，不勾的话后面命令用不了）
- 然后点「Install Now」一路装到底

**验证安装：**

装完后打开命令提示符（按 `Win+R`，输入 `cmd`，回车），输入：

```bash
python --version
```

如果看到 `Python 3.x.x` 就说明装好了。

---

### 2. 下载项目

有两种方式：

**方式一：Git 克隆（推荐）**

```bash
git clone https://github.com/dap33/wechat-jump-pc.git
cd wechat-jump-pc
```

**方式二：下载 ZIP**

点击 GitHub 页面上绿色的「Code」按钮 →「Download ZIP」，解压到你喜欢的文件夹。

---

### 3. 安装依赖

打开命令提示符，进入项目文件夹，执行：

```bash
pip install -r requirements.txt
```

这个项目只需要一个第三方库 Pillow（用于截图和图像处理），tkinter 是 Python 自带的，不需要额外安装。

看到 `Successfully installed` 就说明装好了。

---

### 4. 打开游戏

在电脑微信中打开「跳一跳」小程序：

1. 打开电脑版微信
2. 在聊天列表顶部搜索框搜「跳一跳」
3. 点击进入小游戏
4. 点「开始游戏」，让游戏进入可以玩的画面

> **注意**：游戏窗口不要最小化，不要被其他窗口挡住。

---

### 5. 启动辅助

在项目文件夹中打开命令提示符，输入：

```bash
python gui_app.py
```

这时会弹出一个深色主题的窗口，这就是辅助工具的主界面。

---

### 6. 选择窗口

界面左侧最上方是「游戏窗口」区域：

1. 看一下下拉框里有没有带「微信」或「跳一跳」字样的选项
2. 选中游戏窗口（通常会自动选中第一个疑似微信的窗口）
3. 右侧预览区会显示游戏画面，确认是你正在玩的跳一跳

如果找不到对应窗口，点「刷新」按钮重新扫描。如果没有自动捕获到画面，点击「刷新」重新枚举。

---

### 7. 调参数

两个关键参数在界面左侧「跳跃参数」区域：

**按压系数**（最重要）

- 控制鼠标按下多久 = 跳多远
- 默认 `2.000`，范围 `0.5 ~ 5.0`
- 如果总是跳近 → 把滑块往右拉
- 如果总是跳远 → 把滑块往左拉

**棋子头部直径**

- 影响像素距离到物理距离的换算比例
- 会根据你选择的游戏窗口自动估算
- 一般不需要手动改，除非反复调按压系数都解决不了

> **建议**：先用默认参数跳几把，看情况再微调。参数实时生效，不用停。

---

### 8. 开始跳跃

点击绿色的 **「▶ 开始跳跃」** 按钮。

程序会：
1. 截取游戏画面
2. 识别棋子和目标（右侧预览会显示红蓝标注）
3. 计算距离和按压时间
4. 自动按下鼠标跳跃
5. 重复以上步骤

右侧下方的「运行日志」区域会实时显示每次跳跃的距离和按压时间。

按红色「■ 停止」按钮可以随时停。按「⏸ 暂停」可以临时暂停。

调好参数后点「💾 保存配置」，下次启动直接沿用，不用重新调。

## 参数该怎么调

记住一个口诀：**跳近加系数，跳远减系数。**

| 现象 | 操作 |
|------|------|
| 每次都不够，差一点 | 按压系数 +0.1 试试 |
| 每次都不够，差很多 | 按压系数 +0.3~0.5 |
| 每次都过头 | 按压系数 -0.1 试试 |
| 时准时不准 | 检查游戏窗口有没有被遮挡 |
| 拉到最大还是不够 | 减小「棋子头部直径」5~10px |

## 常见问题

**Q: 提示「No module named 'PIL'」怎么办？**

重新执行 `pip install -r requirements.txt`，确保安装过程没有报错。

**Q: 提示「No module named 'tkinter'」怎么办？**

tkinter 是 Python 标准库。如果你用的是某些精简版 Python（如 miniconda 某些版本），可能需要重装完整版 Python。

**Q: 找不到微信窗口？**

1. 确保微信窗口没有被最小化
2. 点「刷新」按钮
3. 如果微信小程序是独立窗口模式（从微信主窗口分离出来的），找窗口标题带「小程序」的

**Q: 识别不准，棋子/目标定位错误？**

1. 确保游戏画面完整可见，没有被遮挡
2. 确认选的窗口是正确的游戏画面
3. 游戏背景颜色如果变化太大（如夜间模式）可能影响识别

**Q: 跳跃距离偏差很大？**

参考上面「参数该怎么调」部分。核心就是调按压系数。

## 原理简介

1. **截图** — `PIL.ImageGrab` 截取游戏窗口
2. **找棋子** — 按紫色像素范围 (R:50-60, G:53-63, B:95-110) 扫描，取底部中心
3. **找目标** — 在棋子对面半边扫描颜色突变边缘，用对称中心推算 Y 坐标
4. **算距离** — 棋子和目标的欧几里得距离
5. **算时长** — `(-945 + sqrt(945² + 4×105×36×实际距离)) / (2×105) × 1000`，再乘按压系数
6. **模拟跳跃** — `SendInput` API 在游戏区域内随机位置模拟鼠标长按
